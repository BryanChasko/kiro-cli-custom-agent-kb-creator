# Contributing to KB-CREATOR Agent

## Development Workflow

1. Clone the repository
2. Follow the implementation plan in `steering-agent-plan.md`
3. Create steering documents in `~/.kiro/steering/`
4. Build agent configuration in `~/.kiro/agents/`
5. Test each task before moving to the next

## Testing

- Verify agent loads without errors
- Test Firecrawl MCP integration
- Validate knowledge base operations
- Check deduplication logic
- Verify semantic indexing

## Documentation

Update `steering-agent-plan.md` with progress as tasks are completed.
